# Apple_Silicon_Torch

## Intro:
This repo to utilize the GPUs in MacBooks with Apple Silicon M1~M3.


## Creating the virtual environment:

```
python3 -m venv .venv
source .venv/bin/activate
pip install -U pip
```

## Installing PyTorch:
```
pip install torch torchvision torchaudio
```
